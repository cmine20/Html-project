    Calculate 4 numbers

    This is a Calculator java program that allows the user to perform basic arithmetic operations on four numbers. It reads input from the user using the Scanner class and runs in a loop until the user decides to exit. The program prompts the user to enter four numbers and an operator, and then performs the corresponding operation on the numbers.     Receipt with 4 products

    This is a Receipt java program that simulates a receipt generation system for a store. It prompts the user to enter their name and address and the quantities of four products they want to purchase. It calculates the total cost of each product and the overall total cost and displays the receipt on the console. The receipt includes the names and prices of the products, the quantities purchased by the user, and the total cost of each product and the overall total cost. The program reads input from the user using the Scanner class.

    Receipt v.2

    This is a Receipt v.2 java program that allows the user to enter the details of products they want to purchase, calculates the total cost of the products, including VAT, and displays a receipt on the console. The program prompts the user for the number of products they want to purchase, the name, price, and quantity of each product, and their payment amount. It calculates the total cost of each product by multiplying the price by the quantity, calculates the VAT on the total amount using a rate of 12%, and calculates the change the user will receive by subtracting the total amount from the payment amount. The program displays the receipt on the console, including the details of the products purchased, the total amount, VAT, payment, and change. The program runs in a loop until the user decides to exit.
